# .gitignore文件配置

通过配置* .gitignore *文件，可以指定git管理所需要的文件
忽略代码编译过程中产生的额外文件

## 配置方法

## 配置参考

[GitHub](https://github.com/github/gitignore/, "github 推荐")

## example

### c++配置

```bash
# Prerequisites
*.d

# Compiled Object files
*.slo
*.lo
*.o
*.obj

# Precompiled Headers
*.gch
*.pch

# Compiled Dynamic libraries
*.so
*.dylib
*.dll

# Fortran module files
*.mod
*.smod

# Compiled Static libraries
*.lai
*.la
*.a
*.lib

# Executables
*.exe
*.out
*.app
```
